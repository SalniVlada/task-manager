# Task manager

Проект был написан в рамках тестового задания с использованием Vue 2.

## Запуск проекта

Скопируйте себе репозиторий проекта.

В терминале установите зависимости для проекта

```
npm install
```

Запустите проект с помощью команды

```
npm run serve
```
